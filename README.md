django-project-teplate
=======================

a django project template

Environment
================
* python 2.7
* django 1.6

Installation
=================

# start project

```sh
django-admin.py startproject {{ project_name }} --template=django-project-teplate
```

# start app

```sh
django-admin.py startapp --template=django_project_template/apps/app1 {{ app_name }}
```
